# @tap-payments/apple-pay-rn

ReactNative wrapper for apple pay button

## Installation

```sh
npm install @tap-payments/apple-pay-rn
```

## Usage

```js
import { multiply } from '@tap-payments/apple-pay-rn';

// ...

const result = await multiply(3, 7);
```
